"""Build model-ready tables for the medical recommendation module.

"""Train and evaluate a baseline ML medication ranking model.

The model treats each row in patient_condition_medication.csv as a candidate:

    patient + condition + medication -> probability medication is prescribed

It trains a simple logistic SGD classifier and evaluates the output as a
ranking problem within each patient-condition group.
"""

    features, then return top-k medications.

This script works with the existing processed CSV:

    Datasets/processed/patient_condition_medication.csv
